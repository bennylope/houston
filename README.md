# Houston: help launching services on Mac OS

Houston is a port of the popular
[lunchy](https://github.com/eddiezane/lunchy/) program to Go.

### Supported commands

  - ls [pattern]

### Unsupported commands

  - start [pattern]
  - stop [pattern]
  - restart [pattern]
  - status [pattern]
  - install [file]
  - uninstall [pattern]
  - show [pattern]
  - edit [pattern]
